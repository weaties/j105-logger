Tag Manager

Convention. Tags capture human-judgment racing events (close crossings, protests, places gained/lost, rollings). Conditions like light or heavy air are not tags — they are queries against the wind data.

Directional pairs are intentional. rolled-them and got-rolled are separate tags so “every moment we got rolled” remains a filterable query. Do not merge directional pairs — collapsing them destroys the query. Same for pinned-them / got-pinned, we-protested / got-protested, places-gained / places-lost, etc.

Colors are load-bearing. green = us-good, red = us-bad, purple = rules, grey = gear/crew incident, red/amber = collision severity. Keep new tags in the convention where it applies.

Counterparty is not a tag. Which other boat was involved lives on the bookmark itself, not as a per-boat tag — otherwise the tag list drifts into half-boats per season.
